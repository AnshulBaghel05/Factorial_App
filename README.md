# 📱 Factorial App

A simple Android application built using **Kotlin** and **Android Studio** that calculates the **factorial of a number** entered by the user.

This project is designed for beginners to understand:

* Android UI Design using XML
* User Input Handling
* Button Click Events
* Basic Kotlin Logic
* Android Activity Structure

---

# ✨ Features

* 🔢 User can enter any number
* 🔁 Calculates factorial instantly
* ⚡ Fast and lightweight app
* 📱 Simple and beginner-friendly UI
* 🎯 Easy-to-understand Kotlin code

# 🧠 What is Factorial?

Factorial of a number is the multiplication of all positive integers less than or equal to that number.

### Example:

```bash
5! = 5 × 4 × 3 × 2 × 1 = 120
```
